# Multiplix

App d'apprentissage des tables de multiplication pour enfants, basée sur la recherche en sciences cognitives.

**https://isc.github.io/multiplix/**

## Principes

- **Répétition espacée** — Système de Leitner à 5 boîtes
- **Anti-interférence** — Faits similaires séparés dans le temps
- **Entrelacement** — Mélange de tables à chaque séance
- **Compréhension d'abord** — Grille de points avant mémorisation

## Stack

- React + TypeScript + Vite
- PWA (installable, fonctionne hors-ligne)
- localStorage (pas de backend)
- GitHub Pages

## Dev

```bash
npm install
npm run dev
```
